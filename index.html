    Welcome

    This is the official open source home for StruxureWare Plant Operation Ampla from Schneider Electric. With a community of talented engineers building many useful tools for Ampla we decided it was time to help consolidate these efforts and create a central place that can be called home for these projects.

    Projects

    You can find the code and start contributing by visiting our GitHub site.

    • Project Analysis

      Generates a detailed and interactive HTML report from an Ampla project XML export.

    • Security Analysis

      Provides a HTML report detailing the roles, security scopes and role assignments configured in an Ampla project.

    • Project to B2MML

      Use Ampla's Master Data Management functionality to detect changes in the Ampla project.

    • Ampla Data MVC

      ASP.NET MVC project that will use the Ampla Data Webservice with a model based repository pattern.

    • Project-Diff

      A set of scripts to assist in comparing the difference between two projects.

    • Ampla Log Reader

      Read and summarise the various Ampla Log files such as Wcf Logs, Production Analyst Logs and the Windows Event Log.
